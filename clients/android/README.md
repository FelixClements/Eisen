# Android Client

Native Android client for Eisen. Stack: Kotlin + Jetpack Compose + Room + WorkManager.

- minSdk 34 (Android 14)
- targetSdk 36
- compileSdk 37
- JDK 21, Kotlin 2.3.10, AGP 9.2.1

Secure storage: Android Keystore + EncryptedSharedPreferences/Tink for key material; encrypted Room/SQLCipher for task data.

Owner: project owner / mobile
